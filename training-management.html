Менеджмент тренировок

Протестировать приложение
Source code на Github

REST API реализация системы менеджмента тренировок.
Существуют два вида пользователей - администраторы и обычные пользователи.
Администраторы могут создавать/изменять/удалять пользователей, а также их данные.
Пользователи могут регистрироваться в приложении. После успешной аутентификации им становится доступна возможность работать со своим профилем, создавать/изменять/удалять персональные виды упражнений и данные о проведенных тренировках.
