Агрегатор ключевых навыков

Протестировать приложение
Source code на Github

Данное приложение составляет список ключевых навыков для указанной пользователем профессии.
Информация о вакансиях берется с портала Head Hunters Russia.

Пользователь вводит название интересующей его профессии.
Дополнительно пользователь может указать город, в котором необходимо осуществлять поиск вакансий. В случае, если пользователь не указал город, поиск происходит по всем городам России.
Также пользователь может указать количество вакансий (первые 100, 500 или 2000 из выборки Head Hunters Russia), для сбора ключевых навыков.
Выбранное количество вакансий существенно влияет на время составления списка навыков.

После анализа выборки вакансий пользователю представляется список ключевых навыков для выбранной им профессии с указанием (в процентах) количества вакансий, требующих его знание, от общего числа проанализированных вакансий.
Результат агрегации ключевых навыков по профессии за конкретную дату и размер выборки сохраняется в базу данных. Таким образом, если другой пользователь в тот же самый день сделает запрос по такой же профессии в том же городе, с аналогичным размером выборки, результат будет взят из базы данных и представлен мгновенно.
