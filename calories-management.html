Java Enterprise проект с регистрацией/авторизацией и правами доступа на основе ролей (USER, ADMIN).
Существуют два вида пользователей - администраторы и обычные пользователи.
Администраторы могут создавать/изменять/удалять данные пользователей, а пользователи - управлять своим профилем и данными (едой).
Взаимодействие с приложением осуществляется через UI (по AJAX) и по REST интерфейсу с базовой авторизацией.
Возможна фильтрация еды по датам и времени.
Цвет записи таблицы еды зависит от того, превышает ли сумма калорий за день норму (редактируемый параметр в профиле пользователя).
Весь REST интерфейс покрывается JUnit тестами, используя Spring MVC Test и Spring Security Test.
