Система голосования за рестораны

Протестировать приложение
Source code на Github

REST API реализация системы голосования за рестораны.
Существуют два вида пользователей - администраторы и обычные пользователи.
Администраторы могут создавать/изменять/удалять данные ресторанов, а также создавать/изменять/удалять ланч-меню дня для каждого ресторана.
Все рестораны каждый день предоставляют свои ланч-меню дня. Администраторы добавляют их в приложение.
Пользователи могут просматривать информацию о ресторанах, а также предлагающиеся в них ланч-меню дня.
Пользователи могут регистрироваться в приложении. После успешной аутентификации им становится доступна возможность работать со своим профилем, а также голосовать за ресторан, который они хотели бы посетить сегодня.
Пользователь может голосовать только один раз в день. Если пользователь проголосовал за ресторан, а потом решил изменить свой выбор - он может это сделать, но только до 11 часов дня.
